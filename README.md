# \# Allatoona Water Quality Model

# 

# The goal of this project is to develop a model of in-pool and downstream water quality at Allatoona Dam, focusing initially on dissolved oxygen (DO), discharge, and reservoir elevation for the Etowah River tailwater.

# 

# This repository will include:

# \- Daily datasets (USGS DO \& discharge, USACE elevation)

# \- R scripts for data retrieval, cleaning, and joining

# \- Statistical and machine-learning models (linear regression, GAM, etc.)

# \- Visualizations and model diagnostics
